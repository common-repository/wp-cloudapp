== Description ==

This Plugin let you upload images and files directly via your admin-control-panel to GetCloudApp.com!

For support visit the [support-thread on CloudApp-board](http://support.getcloudapp.com/discussions/third-party-applications/3-wp-cloudapp)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you create a post now you will see a box under the text-area.
